storetma-test - Requirements

1. Required software
- Git
- Node.js 16.20+ (recommended: Node.js 18 LTS or 20 LTS)
- npm 8+
- Optional on Windows: WSL2

2. Required environment variables
- DATABASE_URL=postgresql://USER:PASSWORD@HOST:PORT/DB?sslmode=require
  or
- POSTGRES_URL=postgresql://USER:PASSWORD@HOST:PORT/DB?sslmode=require
- STRIPE_SECRET_KEY=sk_test_xxx

3. Project dependencies (installed by npm install)
- next ^15.5.0
- react ^18.3.1
- react-dom ^18.3.1
- @neondatabase/serverless ^1.0.2
- @vercel/postgres ^0.10.0
- vitest ^0.34.6

4. Install steps
- npm install

5. Run project
- npm run dev

6. Run tests
- npm test

7. Notes
- This is a Node.js / Next.js project, so use package.json for dependency installation.
- requiredment.txt is only a quick setup reference file.
- For full setup details, read:
  - docs/setup-a-to-z.md
  - Architecture.md
